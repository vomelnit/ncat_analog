Usage: ncat_analog [options]

Options:
-u, --udp                        Use UDP instead of default TCP
-p, --port port                  Specify source port to use
-s, --source addr                Specify source address to use
-l, --listen                     Bind and listen for incoming connections
-r, --read-from-file <filename>  Transmit data from file
-w, --write-to-file  <filename>  Write received data into file
-h, --help                       Display this help screen
